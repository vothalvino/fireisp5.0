# Badge

Compact status pill for operational states (paid, overdue, active, …). Soft-tinted, fully rounded, tokens only.

- **tone**: `neutral` · `success` · `danger` · `warning` · `accent`. Default `neutral`.
- Children are the label text.

```tsx
<Badge tone="success">Paid</Badge>
<Badge tone="danger">Overdue</Badge>
```
